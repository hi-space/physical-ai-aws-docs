# Isaac Sim 환경 구축

Isaac Sim은 로봇 시뮬레이션과 합성 데이터 생성에 많이 씁니다. AWS에서는 보통 **GPU 인스턴스** 위에 띄우고 원격으로 접속합니다.

### AWS에서의 대표 실행 형태

* 단일 노드 개발: EC2 GPU 1대.
* 배치 시뮬/합성 데이터: 여러 노드로 수평 확장.
* 쿠버네티스 운영: EKS로 잡 단위 실행.

### 운영 설계 포인트

* 렌더링 워크로드는 GPU가 필수인 경우가 많습니다.
* 결과물(비디오/라벨/로그)은 S3에 모으는 게 편합니다.
* 시나리오 정의는 코드로 버전 관리하세요.



NVIDIA Isaac Lab은 Isaac Sim의 공식 로봇 학습 프레임워크로, 강화 학습 (Reinforcement Learning), 모방 학습 (Imitation Learning) 등을 위한 API와 예제를 제공합니다. 이 프레임워크는 다양한 워크플로에서 작업을 설계할 수 있는 기능을 제공하며, 최신 시뮬레이션 기능을 활용하여 로봇 학습 환경을 쉽고 효율적으로 구축할 수 있는 모듈식 설계를 지원합니다.
